Вместе с отделом маркетинга мы подготовили список гипотез для увеличения выручки. Будем приоритизировать гипотезы, запускать A/B-тест и анализировать результаты.



## Стек

* pandas
* matplotlib
* numpy
* scipy
* plotly

## Результат

Провел приоритизацию гипотез для отдела маркетинга интернет-магазина для увеличения выручки, а так же произвел оценку результатов A/B-тестирования различными методами.
