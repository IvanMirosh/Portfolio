## Контекст задачи

Компания предлагает два тарифных плана: «Смарт» и «Ультра». Чтобы скорректировать рекламный бюджет, коммерческий департамент хочет понять, какой тариф приносит больше денег.

Нам предстоит сделать предварительный анализ тарифов на небольшой выборке клиентов. В нашем распоряжении данные 500 пользователей: кто они, откуда, каким тарифом пользуются, сколько звонков и сообщений каждый отправил за 2018 год. Нам необходимо проанализировать поведение клиентов и сделать вывод — какой тариф лучше.



## Стек

* pandas
* matplotlib
* numpy
* scipy
* plotly

## Результат

Проанализировали два тарифных плана, предлагаемых компанией, и поведение клиентов и выделили наиболее перспективный тариф.
